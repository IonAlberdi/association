## Navigation

## Installation
Before you can start:
* Install node.js and npm
* Install mongodb
* start mongodb
* download this package

To install all needed dependencies run
```bash
    npm install
```
next to the `package.json`. Then simply run
```bash
    sudo node index.js
```
to start the server.

## Run

Open a browser an go to:

http://localhost:7000/Triggers/
